# tfernandev

Portfolio personal — sitio estático (`index.html` + `assets/`).

## Ver en local

Abrí `index.html` en el navegador o serví la carpeta con cualquier servidor estático.

## Publicar en GitHub Pages

Repositorio → **Settings** → **Pages** → **Build and deployment**: *Deploy from a branch* → rama `main`, carpeta `/ (root)`.

La URL será: `https://tfernandev.github.io/tfernandev/`
