// freelance 01

OurClub — Members web (bookings & payments)

Context

Subscription product for sports clubs: members, payments, notices, event/ticketing. Pre-existing .NET deployment; scope was additional self-service and APIs without a full platform replacement.

Scope

Members web and backend integration: multi-tenant configuration per club (branding, menus, digital card), SSO, .NET 6 middleware with versioned REST and Swagger. Integration with Web API, MVC backoffice, and ASP.NET on .NET Framework. Events, ticketing, and turnstile-oriented flows in a separate ASP.NET Core API.

Outcomes

Self-service and payment modules integrated with the existing system. New capabilities exposed through versioned, documented APIs. Dapper and direct SQL on high-throughput paths; EF Core where the domain model applied.

Stack

PostgreSQL · .NET 6 · Dapper on high-throughput paths · EF Core · Npgsql · ASP.NET Core · Web API / MVC (.NET Framework) · JWT/OAuth SSO · API keys between services.

PostgreSQL .NET 6 Dapper ASP.NET Core EF Core Legacy .NET (Web API / MVC) SSO · JWT / OAuth

// freelance 02

OurClub — Fan app & access control

Context

Two mobile clients on one product: member app (digital card, content, SSO) and staff app for venue access control. Variable connectivity at venues; offline-capable access validation and synchronization.

Scope

Fan: Expo / React Native app — SSO via expo-auth-session and deep links, QR membership card, club-specific theming, EAS builds for Android and iOS. Access: Same stack — expo-camera to scan codes, validation through the .NET middleware with API keys; local expo-sqlite rosters per event, entry log, and sync when the network returns (NetInfo).

Outcomes

Access validation with intermittent connectivity; SSO consistent with the web product. SQLite data scoped per event; shared navigation and authentication across both apps.

Stack

Expo ~54 · React Native 19 · React Navigation · expo-auth-session · deep links · expo-camera · Axios → middleware · expo-sqlite · NetInfo · Paper / Material-style UI · EAS.

Expo ~54 React Native 19 React Navigation expo-auth-session expo-camera expo-sqlite EAS
